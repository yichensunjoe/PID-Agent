# Issue #1：属性检查器切片

本切片补齐网页中的高频精确编辑能力。所有修改仍通过同一个 `apply_transaction` 服务和 revision 乐观并发提交，不直接修改画布本地状态。

## 单元素属性

选中一个元素后，右侧“属性”标签页根据类型显示可编辑字段：

- `symbol`：标签、位置、宽高、旋转角度、内部名称和样式；
- `junction`：标签、位置、节点半径和样式；
- `connector`：process tag、路径模式和样式；
- `text`：文字内容、位置、字号、对齐和样式；
- `line`：起点、终点和样式；
- `rectangle`：位置、宽高、圆角和样式；
- `circle`：圆心、半径和样式；
- `polyline`：闭合状态和样式。

符号旋转提供 `−90°` 和 `+90°` 快捷按钮，也可以直接输入任意角度。修改符号位置、尺寸或旋转后，服务端继续负责刷新与该符号绑定的 connector 端点。

## Connector 语义检查

选中 connector 时，属性面板只读显示：

- Source 的 `element_id.port_id` 或自由端点坐标；
- Target 的 `element_id.port_id` 或自由端点坐标；
- 当前折点数量；
- 当前 routing 模式。

这样用户可以在网页中确认管线是否真正绑定到设备端口或 junction，而不是只在视觉上重叠。

## 样式

单元素支持修改：

- 线条颜色；
- 填充颜色；
- 线宽；
- 透明度；
- 虚线数组，例如 `8 4`。

多选时显示公共样式。混合值保持为空，提交时只修改用户填写的字段；“设为实线”可批量清除虚线。

## 交互

右侧区域分为“属性”和“Agent”两个标签页。画布出现选择时自动切换到“属性”，Agent 的自定义模型 API 配置和生成能力保持不变。

属性提交失败时显示字段或服务端错误；成功后使用返回的完整文档和新 revision 更新画布。外部 MCP 修改检测机制继续保留。

## 验收

- 网页可将阀门旋转为 `-90°`，无需 MCP；
- 可修改设备标签而无需删除重建；
- 可精确输入位置、尺寸和常用图形几何参数；
- 可修改文字内容、字号和对齐；
- 可查看 connector 的 source/target 真实绑定；
- 多选可在一个原子事务中批量修改样式；
- 属性修改后 revision 增加，撤销/重做继续有效。
